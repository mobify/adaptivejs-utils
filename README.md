Selector Utils
================

Selector utility functions that can be selectively included in your Adaptive.js builds.

[![Circle CI](https://circleci.com/gh/mobify/selector-utils.png?style=badge&circle-token=8fbb0878d31fb6260af4d0518e61760df7eaa7a2)](https://circleci.com/gh/mobify/selector-utils)

##Available Utilities

String Utilities

- `$.capitalize`

Selector Extensions

- `$('.element').removeStyle()`
- `$('.element').scrollTo(options)`

Static Selector Extensions

- `$.scrollTo($element, options)`
